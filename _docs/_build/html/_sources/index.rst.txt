Synth Pack ドキュメント
================================

概要
--------
このプロジェクトは、Pythonで実装された高機能なソフトウェアシンセサイザーです。
基本的な波形生成から、モジュラーシンセ、グラニュラー合成まで、
様々な音声合成手法を提供します。

機能一覧
------------
* 基本波形オシレーター (正弦波、矩形波、のこぎり波など)
* FM合成
* グラニュラー合成
* モジュラーシンセシステム
* 各種エフェクト処理 (ディレイ、リバーブ、コーラスなど)
* フィルター処理 (ローパス、ハイパス、バンドパス)
* ADSRエンベロープ
